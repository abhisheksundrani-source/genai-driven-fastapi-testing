# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-20
### Added
- Initial FastAPI backend with `/health` and `/hello/{name}` endpoints.
- Unit tests using Pytest for backend validation.
- Robot Framework test suite for API health check.
- Data-driven test setup using JSON/YAML files (no database required).
- Dockerfile and docker-compose for containerized deployment.
- GitHub Actions CI/CD pipeline for linting, unit tests, and Robot Framework execution.
- Project documentation with README.md and repository structure.
- MIT License for open-source compliance.
